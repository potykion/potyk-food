# Борщ

## Изи версия

### Ингредиенты

- [MAGGI® Заправка для борща](https://www.maggi.ru/products/maggi-zapravka-dlya-borscha)
- Говядина голяшка на кости - 400г
- Картофель - 250г
- Капуста - 200г

#### Подача

- Укроп
- Сметана

### Приготовление

- Мясо залить 2.5л воды, вскипятить, снять пену, оставить на медленном огне (~80℃) на 2 часа
- Нарезать картофель, капусту, добавить в бульон, варить 10-15 мин
- Добавить заправку на 5 минут
- Подаем со сметаной, укропом
