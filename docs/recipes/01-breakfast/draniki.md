# Драники 


## Ингредиенты

- Картофель — 4–5 шт. (крупных)
- Лук репчатый — ½
- Мука — 1–2 ст. ложки
- Соль, перец, растительное масло для жарки

## Приготовление

1. Натрите картофель и лук на мелкой или средней тёрке. 
2. Отожмите массу руками или через марлю — чем меньше жидкости, тем хрустящее драники.
3. Добавьте муку, соль, перец, перемешайте.
4. Жарьте на раскалённой сковороде с маслом по 3 минуты с каждой стороны до золотистой корочки.

## Топпинги 

- Сметана + опц. укропыч 
- Авокадо + пашотыч
- Окорок 
- Рыбков красни 
- кремчиз + кимчиз
